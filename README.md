## 自助洗车小程序

### 开始

安装依赖包

```
npm install
```

点击微信开发者 IDE，选择 ` 工具 / 构建 Npm`

### 目录说明

```
|-- miniprogram # 小程序源码目录，即 src
    |-- miniprogram_npm # 构建 npm 目录
    |-- app.json # 页面配置
    |-- app.scss # 全局样式
    |-- app.ts # 入口文件
    |-- sitemap.json # 站点爬虫配置
    |-- apis # 接口
    |-- assets # 静态资源
    |-- components # 全局组件
    |-- config # 配置
    |-- pages # 页面
    |-- utils # 工具函数
|-- typings # ts 类型声明
|-- eslintrc.js # eslint 规则
|-- package.json # npm 配置
|-- project.config.json # 项目开发配置
|-- project.private.config.json # 项目开发配置，私有
|-- tsconfig.json # ts 配置
```
